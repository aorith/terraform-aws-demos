# cidr-listings with ECS Fargate

A demo showcasing a deployment of [cidr-listings](https://github.com/aorith/cidr-listings) in AWS.

- ECS with Fargate
- RDS using PostgreSQL
- ALB
- [SOPS](https://aorith.github.io/posts/secrets-sops/)
